### Portfolio Website

A clean, modern personal portfolio site to showcase projects, experience, and skills. It’s designed to be fast, responsive, and easy to tailor to your own story.

---

### Project Overview

This portfolio includes:
- **Hero / Intro**: Concise introduction and clear call-to-action.
- **Projects**: Selected work with descriptions, tech used, and live/code links.
- **About**: Short bio, background, and key strengths.
- **Skills**: Snapshot of tools, languages, and frameworks.
- **Contact**: Simple ways to reach out (email, LinkedIn, optional form).
- **Responsive Layout**: Looks good on desktop, tablet, and mobile.

### Running the Project Locally

1. **Clone the repository**

```bash
git clone https://github.com/DR2929/Portfolio.git
cd Portfolio
```

2. **Install dependencies**

```bash
npm install
# or
yarn install
pnpm install
```

3. **Start the development server**

```bash
npm run dev
# or
yarn dev
pnpm dev
```

4. **Open the site**

- Visit `http://localhost:3000` (or the port shown in your terminal).

5. **Create a production build (optional)**

```bash
npm run build
# or
yarn build
pnpm build
```
